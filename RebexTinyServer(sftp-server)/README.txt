====================================
 Rebex Tiny SFTP Server
====================================

A minimalist SFTP server for testing purposes.

https://labs.rebex.net/tiny-sftp-server/


## Usage
-------------------------------------
Just run the executable and press the Start button.
Configure it by editing RebexTinySftpServer.exe.config if needed.

## Customization
-------------------------------------
Partial C# source code of Tiny SFTP Server is available, making
it possible to customize it to better suit your requirements:
    https://labs.rebex.net/tiny-sftp-server/

The source code needs Rebex File Server .NET component to compile.
Download a trial version or purchase a license:
    https://www.rebex.net/file-server/
